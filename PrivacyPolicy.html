Privacy Policy

Last updated: March 18, 2025

1. Information We Collect

We collect and analyze public TikTok data through the TikTok API, including:

  • Trending Videos (`video.list`): Video engagement metrics like likes, views, and shares.
  • User Statistics (`user.info.stats`): Public TikTok user metrics, including follower counts and total likes.

2. How We Use This Data

The data is used solely for research and analytics purposes, helping users identify trending content and influencer engagement metrics.

3. Third-Party Services

We use the TikTok API to collect public data. We do not sell or distribute collected data to third parties.

4. User Data Rights

Since we only process publicly available TikTok data, no personally identifiable information (PII) is stored.

5. Data Security

We take precautions to prevent unauthorized access to collected data. However, we do not store personal TikTok user credentials.

6. Changes to This Policy

We may update this policy periodically to align with TikTok API changes.

For privacy-related concerns, contact us at support@yourwebsite.com.
